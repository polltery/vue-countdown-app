## How to run?

- Install node.js and npm (optional, you can also use your server to serve the pages).
- Install `http-serve`.
```
npm i -g http-serve
```
- Run the project
```
http-serve ./
```

## Libraries used

- Bootstrap 5.0.2 : https://getbootstrap.com/docs/5.0/getting-started/introduction/
- Vue 2.0.0 : https://vuejs.org/v2/guide/
